### 一、 简介
　　此仓源代码来自笔者对技术论证以及探索开发的实验代码, 非开源项目, 仅供学者参考与交流。旨在于提供针对性的技术理解与学习。

### 二、 结构
　　①主仓库名为Demo, 根目录附有README.MD自诉文件; 

　　②内分多个子模块, 每个子模块意为针对某技术和某理论实验论证的代码, 并在其根目录同样附有对子模块的README.MD自诉文件;

　　③若实验项目本身结构复杂为了清晰的呈现技术难点给广大读者, 可能会根据情况再细分子模块。
 
　　④以下以Aio的异步通讯为例:

　　</pre>
